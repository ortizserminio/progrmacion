Proyectos de programación.

## To-Do List (`todo.py`)

Gestor de tareas por línea de comandos en Python.

**Uso:**
```bash
python todo.py
```

**Funciones:**
- Ver todas las tareas
- Agregar tarea nueva
- Marcar tarea como completada
- Borrar tarea
- Las tareas se guardan automáticamente en `tareas.json`
